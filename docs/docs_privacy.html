Privacy Policy

frncsrr (“the developer”) crafts experimental, futuristic applications with anonymity as the default stance. No personally identifying information is processed unless you voluntarily provide it for support. This policy clarifies which data is collected, how it is used, and how your rights are respected.

1. Data collected

  • Hashed device class and OS version, strictly for compatibility validation during testing
  • Crash stack traces that are pre-scrubbed to remove identifiers before storage
  • Ephemeral connection metadata when contacting official servers (IP addresses are immediately hashed irreversibly and deleted within 24 hours)

No email addresses, names, phone numbers, or hardware identifiers are collected by default.

2. Purpose of use

Data is used only for the following:

  • Maintaining rollout stability and recovering from outages
  • Prioritizing improvements based on anonymous aggregate statistics
  • Detecting security events and preventing unauthorized access

Only anonymized, aggregated analytics are performed. Data is never sold or shared with third parties.

3. Retention & disposal

Information is retained only as long as it is operationally required. Incident logs are kept for up to 30 days. Hashed connection metadata is purged automatically within 24 hours.

4. Security

All transport uses TLS. Stored data is guarded by row-level access controls, and zero-trust practices reduce the risk of insider leaks. Audit logs intentionally exclude personal information.

5. User choices

Registration or login is never required. If you send optional information (for example in a support email), it is deleted once the request is resolved.

6. Contact

Email francisairry@proton.me for any privacy questions. Anonymous mail services are welcome.
